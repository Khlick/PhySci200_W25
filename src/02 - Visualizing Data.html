Statistical Purpose

To answer a question about a population (of data).

In a perfect world, we would have access to the full population so we would be able to answer the question based on data collected on all the observational units of the population. We would know the value of the true parameter we are interested in.

But usually we just have one sample.

Statistics are used to describe the distribution of the sample and relate it to the distribution of the population.

Types of Hypotheses

Generally, a hypothesis is an assumption about certain characteristics of an object of interest/study. This assumption may or may not be true.

A scientific hypothesis is a specific statement proposed as an explanation for a phenomenon, regardless of truth.

A statistical hypothesis is a specific statement about a characteristic or parameter of the data.

Scientific

  • Is farming salmon cost prohibitive?
  • Does smoking cause lung cancer?
  • Does ketogenic diet improve prognosis for stroke patients?

Statistical

  • Is the average length of farmed salmon greater than wild salmon?
  • Are Diacetyl concentrations and lung cancer cell-division rate correlated?
  • Does ketogenic diet decrease cleaved caspase-12 expression following stroke?
  • General: An assumption that may or may not be true.
  • Scientific: A specific proposal as an explanation for some phenomenon.
  • Statistical: A specific statement about some aspect of the DATA.

These are clear distinctions. We need to be careful that we don't confuse them.

Quantitative Variables

A random variable $X$ is a function that maps outcomes from a sample space $\Omega$ to real numbers.

Mathematically, we write this as:

$$X: \Omega \rightarrow \mathbb{R}$$

There are two types:

Discrete Random Variables

Discrete random variables measure outcomes at distinct or finite points.

Examples include the number of action potentials fired by a neuron or the count of red blood cells in a blood sample.

Continuous Random Variables

Continuous random variables can assume any value within a range of real numbers.

Examples include blood pressure measurements, membrane potentials of neurons, or hormone concentrations in plasma.

Understanding random variables is crucial for analyzing data and making predictions in various fields.

In probability theory, a random variable is a fundamental concept that helps us quantify uncertain outcomes. A random variable is essentially a function that assigns numerical values to each outcome in a sample space. This allows us to work mathematically with probabilities and outcomes, converting qualitative events into quantitative analysis.

Types of Variables

A Response Variable is the metric about which questions are asked, it measures the outcome of the study.

An Explanatory Variable is any metric that can influence, explain, or predict the response variable.

📌 The response variable is usually called the dependent variable, and the explanatory variable is usually called the independent variable.

Data Collection

The population is the entire collection of observational units we are interested in. A sample is a subset of the population from which we record data.

Numerical summaries about a population are called parameters, and numerical summaries calculated from a sample are called statistics.

Describing A Dataset

When describing a dataset, we generally consider the following three questions:

  • What is the general shape of the data?
  • Where are the data values centered?
  • How do the data vary?

These are all aspects of what we call the distribution of the data.

We use some simple arithmetic, which depends (a little) on what the data distribution represents, to describe these aspects of the data.

Recall that our purpose, in statistics, is to gain some insight into the world by taking finite samples.

Shape

When we talk about the shape, we generally mean the shape of the envelope, if a histogram, or the shape of the density function.

General Shape Of A Distribution

  • A symmetric distribution is one in which the left and right hand sides of the distribution are roughly equally balanced.
  • A skewed (asymmetric) distribution is one in which there is no such equal balance. Right-skewness refers to a longer right tail, while left-skewness correspond to a longer left tail.
  • A uniform distribution is a specific symmetric distribution in which all outcomes are equally likely.

Central Tendency

When we talk about the central tendency, we refer to the point where the probability mass, or density, function is greatest.

The Mean

The arithmetic mean is generally calculated as:

$$\bar{x}=\frac{1}{N}\sum_{i=1}^{N}x_i$$

Calculation: $$\begin{aligned} \bar{x} &=\frac{x_{1}+x_{2}+...+x_{n}}{n} \\ &=\frac{\textrm{sum of all the observations}}{\textrm{number of observations}} \end{aligned}$$

Read: "ex-bar", we can see that it represents the sum of a random variable when all values carry equal weights $(\frac{1}{N})$.

We can generalize for unequal weights.

Let $w(x|a)=\sum_i^n x_i==a$ be an indicator of value $a$, and $\sum_{b \in S} w(b|S_x)$ is the sum of all possible weights in the set, $S_x$.

Then $p(x) = \frac{w(x)}{\sum_{b \in S_x} w(b)}$ is the weight of value X.

Thus, the weighted Mean of random variable $X$ is: $$\bar{x}_w = \sum_{i=1}^{N}x_i p(x_i)$$

An indicator function is a function that counts the number of occurrences.

Looks familiar! We also call this the first central moment, the expected value of the random variable.

The Median

The median of an ordered set of data values is:

  • The average of the middle 2 values (for an even number of entries)
  • The middle entry (for an odd number of entries)

mean of $(\frac{n}{2})^{th}$ and $(\frac{n}{2}+1)^{th}$ observations

$(\frac{n+1}{2})^{th}$ observation

Note that the weight of the two values in the middle are equal.

Sometimes referred to as a measure of location as it tells us where the data are.
Median is the value which occupies the middle position when all the observations are arranged in an ascending/descending order. It divides the frequency distribution exactly into two halves. Fifty percent of observations in a distribution have scores at or below the median. Hence median is the 50th percentile.[2] Median is also known as "positional average"
To find the median, your numbers have to be listed in numerical order from smallest to largest, so you may have to rewrite your list before you can find the median.

The Mode

The mode is the value that occurs most often in the dataset. If no value in the dataset is repeated, then there is no mode.

Variance and Standard Deviation

A common measure of data variability is the variance, which measures the squared distance each data point is from the mean, on average.

$s^2 = \frac{\sum(x_{i}-\bar{x})^2}{n-1}$ Variance

$s^2 = \frac{(x_{1}-\bar{x})^2+(x_{2}-\bar{x})^2+...+(x_{n}-\bar{x})^2}{n-1}$

$s^2 = \frac{\textrm{sum of observed squared distance from sample mean}}{\textrm{number of observations - 1}}$

$s^2$ for a sample
$\sigma^2$ for a population

For standard deviation, we use:
$s = \sqrt{s^2}$ and $\sigma = \sqrt{\sigma^2}$.

The standard deviation puts the variance into the same units as the data, providing a measure of how large the average standardized distance from the center is.

Note: The use of $n-1$ in calculating variance helps to ensure that our estimate of the population variance is unbiased and accounts for the extra uncertainty introduced by estimating the population mean from the sample itself.

The larger the variance, the more spread out the data are. However, since variance is in squared units, we often use standard deviation to interpret variability in the original units of the data.

Standard deviation measures the average distance from the mean, in the same units as the data. This makes it easier to understand how spread out the values are around the mean.

QUESTION: Can you think of an assumption or two we have to make if we report the standard deviation?

  • Bias Correction: Dividing by n-1 corrects for the bias of underestimating the population variance when using the sample mean, making the sample variance an unbiased estimator.
  • Degrees of Freedom: Accounting for the loss of one degree of freedom (due to estimating the mean from the sample itself) requires dividing by n-1 instead of n.
  • Increased Accuracy: This adjustment increases the accuracy of the variance estimate, reflecting the true variability in the population more accurately.

But what is "bias"?

Bias

Sampling bias: when a sampling method systematically yields results that are either too high or too low.

May be avoided by using good sampling technique (randomized).

Sampling variation: natural variation in results from one random sample to next.

May be reduced by using a larger sample.

Standard Deviation

Let's consider 2 sets of data, both have a mean of 100

Numbers Mean SD
100, 100, 100, 100, 100 100 0
90, 90, 100, 110, 110 100 10

Set 1: all values are equal to the mean, so there is no variability at all

Set 2: one value equals the mean and other four values are about 10 points away from the mean. So the average distance away from the mean is about 10

Example, procedure for Set 2:

  1. 1. Calculate the sample mean: $\bar{x}=100$
  2. 4. Sum result of step 3 divide by $(n-1)$: $s^2=\frac{400}{5-1}=\frac{400}{4}=100$
  3. 2. Calculate the difference between each value and the mean: [-10, -10, 0, 10, 10]
  4. 5. Square root result in step 4: $s=\sqrt{100}=10$
  5. 3. Square each difference in step 2: [100, 100, 0, 100, 100]

When we take the square root, we are only concerned with the positive square root, but given the assumption of symmetry (normal distribution), we can think of it as plus or minus some variability.

Use SD With Caution

Like the mean, the standard deviation does not cope well with skewed distributions.

Why Are We Squaring Things?

$$s^2 = \frac{\sum(\class{pop}{x_{i}}-\class{danger}{\bar{x}})^2}{n-1}$$

Variance is the sum of the squared distances from each data point to the model.

Squaring the distances ensures that the distances are always positive.

Squaring makes large deviations gigantic and small deviations minuscule and imposes greater weight to larger deviations, which taking the square root doesn't fix.

Why the hell would we want to give more weight to large deviations and reduce weight to small deviations?

The Normal Distribution

The normal (or Gaussian) distribution is a continuous probability distribution characterized by a symmetric, bell-shaped curve.

PDF:

$$f(\class{warning}{x}; \class{info}{\mu}, \class{info}{\sigma}) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}$$
  • $\mu$ = Central Tendency ($E[X]$)
  • 68% of the observations lie within 1 standard "distance" of the center
  • $\sigma$ = Spread ($E[X-E[X]]$)
  • 95% lie within 1.96 standard "distance" of the center
  • $x$ = Specific value of the continuous variable
  • 99% lie within 2.58 standard "distance" of the center

Often denoted $\mathcal{N}(\mu, \sigma^2)$, the normal distribution is special as it underscores the Central Limit Theorem's revelation that sums of independent variables universally converge to this form, regardless of their initial distributions.

The normal distribution stands as a fundamental pillar in statistics. We will cover many other continuous distributions as we go along, but we start with the gaussian.

We talk about the x values being a certain "distance" from the center, mu, in terms of HOW FREQUENT.

Karl Pearson (dwg) called the PDF the "Frequency Curve". Just like the counting we did for discrete distributions, these smooth curves represent the relative count at any given X value. If the area under the curve sums to 1 and represents the sum of all P(X=X), then any point X_i has the probability f(x;mu,sigma).

fragment 1, equation. While it looks quite complicated, there are 2 parameters we are concerned with, mu and sigma

The symmetry and shape of this distribution allows us to think about how extreme data might be. The weights of data lie in the central bell shape, while the probability of taking a value further and further away from 0, in this case, becomes increasingly unlikely.

It turns out that the normal distribution describes the distribution of many natural phenomena. Let's see why that is.

Let's Get MAD

$$\text{MAD}(x) = \text{median}(|\class{pop}{x_{i}} - \class{info}{\text{median}(x)}|)$$

The median absolute deviation (MAD) is a measure of the variability of a dataset. It is calculated by taking the median of the absolute differences between each data point and the median of the dataset.

MAD is a robust measure of variability, meaning it is less affected by outliers than other measures of variability

Extended Measures Of Spread

Ways To Describe Intervals And Ranges

Aside from Variance and Standard Deviation, we can use other measures of spread to communicate properties of the data distribution.

The Range

The range gives you the most basic information about the spread of a dataset. It is calculated by the (arithmetic) difference between the lowest and highest data value.

Percentiles

The $k^{\text{th}}$ percentile is a value in the dataset that has $k\%$ of the data values at or below it and $(100 - k)\%$ of the data values at or above it.

Note:
Here our dataset contains 40 data points. So each data point correspond to $\frac{100}{40}=2.5\%$ of the data.

A "quantile" is simply the fractional position, i.e., $\frac{k}{100\%}$.

Quantiles and Percentiles

  • Sort the data: Arrange all observations in ascending order.
  • Determine the quantile position: Use $Q = \frac{(n+1) \times p}{100}$ for percentiles, where $n$ is the number of observations and $p$ is the percentile.
  • Interpolate if necessary: If $Q$ is not an integer, interpolate between the two closest ranks to find the value.
  • For quartiles: Quartiles divide the data into four equal parts. Use $p = 25, 50, 75$ for the 1st, 2nd (median), and 3rd quartiles, respectively.
  • Special case - Median: The median (2nd quartile) splits the dataset in half. Use $p = 50$.

Actually, this isn't really sufficient for our purposes.

Understanding Quantiles

Quantiles are values that mark where specific proportions of your data fall below.

The quantile function is generally defined as: $\class{success}{Q_{i}(p) = (1 - \gamma)x_{j} + \gamma x_{j+1}}$,
where $j = \lfloor k \rfloor$ for $k = np + m$ and $\gamma = k - j$.

For quantile method 8 (unbiased median), the quantile function is defined as
\(Q_{8}(p) = (1 - \gamma)x_{j} + \gamma x_{j+1}\),
where \(j=\lfloor k \rfloor\), \(m = \frac{p + 1}{3}\), \(k = np + m\), and \(\gamma = k - j\).

Reference: Hyndman and Fan (1996) for a detailed explanation on quantile types and their applications.

Given dataset: \([-0.977, -0.151, -0.103, 0.4, 0.411, 0.95, 0.979, 1.764, 1.868, 2.241]\)

Calculating the median from $Q_8(\class{danger}{0.5})$:

\(Q_{8}(\class{danger}{0.5}) = (1 - \class{pop}{\gamma}) x_{\class{info}{5}} + \class{pop}{\gamma}x_{\class{info}{5}+1} = (1-\class{pop}{0.5}) 0.411 + (\class{pop}{0.5})0.95 = 0.6805\)

Calculating the median from taking mean of middle 2 elements:
$ m = \frac{0.411 + 0.95}{2} = 0.6805$

  • \(n = 10\)
  • \(p = \class{danger}{0.5}\)
  • \(m = \frac{p + 1}{3} = \frac{0.50 + 1}{3} = \frac{1.5}{3} = 0.5\)
  • \(k = np + m = 10 \times 0.50 + 0.5 = 5.5\)
  • \(j = \lfloor k \rfloor = \lfloor 5.5 \rfloor = \class{info}{5}\)
  • \(\gamma = k - j = 5.5 - 5 = \class{pop}{0.5}\)

Quantiles are essential in understanding the distribution of data.

There are several methods or types for quantile calculation, we will use type 8, an unbiased median estimator

What is the data space (units) of the quantile function? The same as the data. That is, we get a location in the dataset where the pth fraction of the number of data points falls below.

How does this relate to the normal distribution? We can use a similar approach and map our data points to the cdf values of the normal distribution. To do that, we need another function.

Interquartile Range

The median divides the data into two equal halves (it is the $50^{th}$ percentile). If we divide each of those halves again, we obtain two additional statistics known as the first (Q1) and third (Q3) quartiles, which are the $25^{th}$ and $75^{th}$ percentiles.

Interquartile range: $\textrm{IQR} = Q3 - Q1$

A value is considered an outlier if it is:

Smaller than $Q1-1.5\times IQR$

or

Larger than $Q3+1.5\times IQR$

Python Code



import numpy as np

data = [-0.977, -0.151, -0.103, 0.4, 0.411, 0.95, 0.979, 1.764, 1.868, 2.241]

Q1 = np.quantile(data, 0.25, method='median_unbiased')

Q3 = np.quantile(data, 0.75, method='median_unbiased') # Type 8

IQR = Q3 - Q1

outliers = [x for x in data if x < Q1 - 1.5 * IQR or x > Q3 + 1.5 * IQR]

print(outliers)

To objectively identify outliers in a dataset, we consider measures of location and spread to determine when a value significantly deviates from typical cases. A common approach involves using quartiles and the interquartile range (IQR). Typically, most data points fall within approximately 1.5 times the IQR from the quartiles. This method helps in assessing whether a value, such as a notably high lifespan, is an outlier by comparing it against this standard.

Outliers

An outlier is an observed value that is notably distinct from the other values in a dataset. Usually, an outlier is much larger or much smaller than the rest of the data values.

Displaying the data: Boxplot

A boxplot is a graphical display of the five number summary for a quantitative variable. It shows the general shape of the distribution, identifies the middle 50% of the data, and highlights any outliers.

A boxplot includes:

  • A box stretching from Q1 to Q3
  • A line that divides the box drawn at the median
  • A line from each quartile to the most extreme data value that is not an outlier. (if no outliers minimum and maximum)
  • Each outlier plotted individually

From Dotplot To Histogram

A dotplot, challenging to construct with overlapping dots for similar, numerous values, can be replaced by a histogram. Histograms aggregate similar values through counts, effectively displaying data distribution.

Process to construct a histogram

1 Define "boundaries" (they form bins)

2 Count the number of elements inside each bin

A histogram lets you discover the underlying frequency distribution (shape) of a set of data.
