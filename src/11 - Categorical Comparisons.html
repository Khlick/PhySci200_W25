Chi-Squared ($\chi^2$)

A statistic that is widely used to describe the difference in distribution patterns of categorical variable(s) is the square of the relative differences of each observed frequencies compared to the expected frequencies. This is known as the Chi-Squared statistic for comparing frequencies of a categorical variable.

$\chi^2=\sum_{i=1}^n\frac{(\mathrm{Observed}_i-\mathrm{Expected}_i)^2}{\mathrm{Expected}_i}$

$\chi^2$ summarizes the discrepancy between observed and expected frequencies:

  • The smaller the overall discrepancy between the observed and expected scores, the smaller the value of the $\chi^2$.
  • Conversely, the larger the discrepancy between the observed and expected scores, the larger the value of the $\chi^2$.

Recall, chi-squared is the statistic we use to sum up the relative differences in COUNTS from corresponding bins of 2 distributions.

Differences Between Bins

Calculating a \( \chi^2 \) statistic is similar to determining normalized squared residuals in a linear regression model, applied to categorical data in histograms. It assesses deviations of observed categories from expected outcomes under a hypothesis.

How significantly does the distribution pattern of observed frequencies differ from the theoretical distribution? This analysis mirrors the evaluation of sum of squares between groups in ANOVA, which measures variance due to different groupings within the data.

Key idea

When we calculate a \( \chi^2 \) statistic to compare distribution patterns of categorical variables, this is called a \( \chi^2 \) goodness of fit test.

Note that here we compare our data to a theoretical distribution, when else did we do that?

$\chi^2$ Goodness Of Fit

When we calculate a $\chi^2$ statistic to compare distribution patterns of categorical variables, this is called a $\chi^2$ goodness of fit test.

$\chi^2=\sum_{i=1}^n\frac{(\mathrm{Observed}_i-\mathrm{Expected}_i)^2}{\mathrm{Expected}_i}$

Rock Paper Scissors
Observed 107 124 150
Expected 127 127 127

$\chi^2=\frac{(107-127)^2}{127}+\frac{(124-127)^2}{127}+\frac{(150-127)^2}{127}$

$\chi^2=3.150+0.071+4.165$

$\chi^2=7.386$

What does a $\chi^2$ of 7.386 mean? Is it a large or a small value?

Follow Up

We found a difference in distribution pattern that may be considered statistically significant.

So, what's next

The $\chi^2$ test tells you whether you have an overall difference between the observed and expected pattern, but it does not tell you which specific groups differ: post hoc tests do.

Note: Because post hoc tests are run to confirm where the differences occurred between groups, they should only be run when you have shown an overall statistically significant difference. Post hoc tests are termed a posteriori tests, that is, performed after the event (the event in this case being the overall analysis).

Follow Up

We found that there is a difference in distribution pattern that can be considered statistically significant ($\text{p-value}\lt 0.05$)

So, what's next

No standard approach to follow, there are many different follow-up approaches possible.

We want to know if one of the hand sign (rock, paper, scissors) is used more often compared to the others so we could play the opposite sign more often to get an advantage.

Now that we know that there is a statistical difference in the distribution pattern of the signs, we could use pairwise analysis of difference in proportions to determine exactly which signs are more often played (i.e.: Tukey post hoc test).

def expandGrid(*vals, doPaste=True, sep=" ", expansion=None, doSort=False):

    output = np.array(np.meshgrid(*vals)).reshape(len(vals), -1).T

    if expansion is not None:

        if len(expansion) != output.shape[0]:

            import warnings

            warnings.warn('Length of "expansion" must be the same as output.')

            return output

        output = np.repeat(output, expansion, axis=0)

    return output
